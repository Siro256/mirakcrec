# mirakcrec

mirakcrec (miracle-see) は [mirakc](https://github.com/mirakc/mirakc) で録画をするためのフロントエンドです。

## 機能

- 複数言語のサポート (日本語 / 英語)
- <span style="color: gray;">番組表 (未実装)</span>
- <span style="color: gray;">番組表からの録画予約 (未実装)</span>
- <span style="color: gray;">録画予約の管理 (未実装)</span>
- <span style="color: gray;">録画済み番組の管理 (未実装)</span>
- <span style="color: gray;">Webhookでの通知 (未実装)</span>
