# mirakcrec

mirakcrec (miracle-see) is the recording frontend for [mirack](https://github.com/mirakc/mirakc).

## Features

- Multiple language support (Japanese / English)
- <span style="color: gray;">TV guide (to be implemented)</span>
- <span style="color: gray;">Schedule a recording from TV guide (to be implemented)</span>
- <span style="color: gray;">Managing schedules (to be implemented)</span>
- <span style="color: gray;">Managing recordings (to be implemented)</span>
- <span style="color: gray;">Webhook notification (to be implemented)</span>
